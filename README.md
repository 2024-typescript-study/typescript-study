# 타입스크립트 스터디

## 📚 스터디 소개
이 스터디는 타입스크립트 관련 서적을 기반으로 진행되는 스터디입니다.
매주 2개의 챕터를 학습하고, 각자의 방식으로 내용을 정리하여 공유합니다.

### ⏰ 진행 방식
- 기간: 2024.11.01 ~ 매주 화요일 오후 8시
- 장소: 디스코드 음성채널
- 내용: 매주 2개 챕터 정리 및 공유

## ✅ 스터디 규칙

1. 2회 이상 불참시 강제 퇴장입니다.
2. 내용을 제대로 준비하지 못한 경우 불참으로 간주합니다.
2. 책의 모든 부분까지 정리를 마칠때까지 스터디는 유효합니다.
